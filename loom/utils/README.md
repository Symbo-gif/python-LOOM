The utils module provides internal helper functions used throughout loom:

- Shape validation utilities
- Type checking helpers
- Internal decorators for operation registration

## Usage

Most utilities are internal. The primary user-facing utilities are accessed through the main `loom` API rather than directly from `utils`.
